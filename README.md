# 紅茶 SNS

チャット機能を備えたシンプルなSNSです。Node.jsとExpressでサーバーを立てています。

## 動作環境

- Node.js 18.x
- npm 9.x

## セットアップ

```bash
git clone https://github.com/username/repository.git
cd repository
npm install
node server.js
```
